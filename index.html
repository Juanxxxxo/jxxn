Lenguajes de Programación

Con el pasar del tiempo la programación ha evolucionado de tal manera que han surgido los "Lenguajes de Programación". Al día de hoy existen muchos lenguajes, pero los más utilizados por los programadores son:

  1. HTML
  2. Java
  3. JavaScript
  4. C++

Siendo "HTML" la base de la programación.
